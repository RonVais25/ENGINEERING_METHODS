GoNature DB Description

Main tables:
- users: staff users and roles.
- parks: park capacity and parameters.
- visitors: visitor personal data.
- reservations: planned visits.
- waiting_list: waiting list requests.
- visits: actual entry and exit records.
- bills/payments: billing and payment data.
- promotions: discount promotions.
- notifications: simulated email/SMS messages.
- generated_reports: report generation audit.
- reservation_cancellations: cancellation and no-show tracking.
